# ignitecli

macOS single-script execution manager for MDM-driven onboarding, migration, and login window workflows.

## Installation

Download the latest `.pkg` from [Releases](https://github.com/headmin/ignitecli/releases).

```bash
sudo installer -pkg ignitecli-0.1.0.pkg -target /
```

## Requirements

- macOS 14+
- [SwiftDialog](https://github.com/swiftDialog/swiftDialog) (v2.6+)
